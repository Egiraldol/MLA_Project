# Bank Marketing - Machine Learning Project

## Descripción
Proyecto de clasificación supervisada para predecir si un cliente aceptará un depósito a término usando el Bank Marketing Dataset (UCI).

## Contenido
- EDA y baseline en notebook
- Reporte en LaTeX (PDF incluido)

## Dataset
UCI Machine Learning Repository:
https://archive.ics.uci.edu/ml/datasets/Bank+Marketing

## Cómo ejecutar
```bash
pip install -r requirements.txt
jupyter notebook
```

## Resultados baseline

- Accuracy: ~0.90
- ROC-AUC: ~0.90
- F1-score (clase positiva): bajo debido al desbalance
